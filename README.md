# web-test

KDN에서 실습으로 만드는 웹사이트 — 개인 소개 페이지.

## 파일 구성

- `index.html` — 메인 페이지 (About / Skills / Projects / Contact 섹션)
- `style.css` — 모던 미니멀 라이트 테마 스타일시트

## 내용 채우는 법

`index.html` 안의 `{{...}}` 자리표시자를 본인 정보로 교체하세요.

| 자리표시자 | 의미 |
|---|---|
| `{{이름}}` | 이름 (브랜드, 타이틀, 푸터에 사용됨) |
| `{{소속_부서}}` | KDN 내 부서/팀 |
| `{{한_줄_소개}}` | 히어로 영역 한 줄 소개 |
| `{{자기소개_본문_1문단}}` | About 섹션 첫 문단 |
| `{{자기소개_본문_2문단_선택}}` | About 섹션 두 번째 문단 (선택) |
| `{{스킬_1}}` ~ `{{스킬_6}}` | 스킬 태그 |
| `{{프로젝트_N_제목/기간/설명}}` | 프로젝트 카드 |

VS Code 등에서 `{{`로 검색하면 모든 자리표시자를 빠르게 찾을 수 있습니다.

## 미리보기

`index.html`을 더블클릭하거나 브라우저에서 직접 열면 됩니다. 별도 서버가 필요 없습니다.
